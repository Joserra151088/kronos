# Manual de Uso para Empleados

## 1. Objetivo
Este manual explica el uso diario de la plataforma `Kronos` para empleados operativos.

## 2. Acceso al sistema
1. Ingrese a la URL del sistema.
2. Capture su correo y contrasena.
3. En caso de ser medico de guardia, seleccione la sucursal correspondiente.
4. Presione `Iniciar sesion`.

## 3. Pantalla principal
En el dashboard el empleado puede ver:
- saludo y perfil;
- estado del dia;
- siguiente registro esperado;
- resumen de actividad;
- accesos a incidencias, notificaciones y perfil.

## 4. Como registrar asistencia
1. Active GPS en su dispositivo o navegador.
2. Ubicarse dentro del perimetro autorizado.
3. Presione el boton de registro.
4. El sistema guardara automaticamente el siguiente evento:
   - entrada
   - salida a alimentos
   - regreso de alimentos
   - salida final

## 5. Reglas importantes
- Debe registrarse desde la ubicacion autorizada.
- El sistema valida geocerca.
- Existe un tiempo de espera entre registros para evitar duplicados.
- Si una marca no puede realizarse correctamente, debe solicitar apoyo o levantar una aclaracion.

## 6. Consulta de registros
En la seccion de registros podra:
- revisar marcas del dia;
- consultar historial;
- ver tiempo trabajado;
- identificar registros manuales o cambios autorizados.

## 7. Incidencias
Use la seccion de incidencias para reportar:
- enfermedad;
- permisos;
- retardos justificados;
- ausencias;
- documentos relacionados.

### Pasos
1. Entre a `Incidencias`.
2. Seleccione el tipo.
3. Capture descripcion y adjuntos si aplica.
4. Envie la solicitud.

## 8. Aclaraciones de horario
Si una marca no fue correcta:
1. Entre al modulo correspondiente.
2. Capture la fecha, tipo de registro y motivo.
3. Envie la aclaracion.
4. Espere la revision del supervisor o area responsable.

## 9. Notificaciones
La campana superior muestra avisos del sistema:
- resolucion de incidencias;
- respuesta a aclaraciones;
- mensajes operativos.

## 10. Perfil personal
En `Perfil` puede actualizar:
- nombre y apellidos;
- telefono;
- correo electronico si el rol lo permite;
- foto de perfil;
- contrasena.

## 11. Recuperacion de contrasena
Si olvida su contrasena:
1. Use la opcion de recuperacion en login.
2. Abra el enlace recibido.
3. Capture su nueva contrasena.

## 12. Buenas practicas
- No comparta su contrasena.
- Verifique que el GPS este activo antes de registrar.
- Revise sus notificaciones.
- Reporte incidencias el mismo dia.

## 13. Soporte
Si no puede ingresar o registrar:
- contacte a su supervisor;
- contacte al area de soporte TI;
- proporcione fecha, hora y descripcion del problema.
